DB2 utilise des tables système


1) Avec spufi, afficher les tables que vous avez créées.

SELECT *  FROM SYSIBM.SYSTABLES  
	WHERE CREATOR = 	'APInn' ; 
	
	
2) DB2 utilise  une table système pour enregistrer le nom des colonnes créées.
SYSIBM.SYSCOLUMNS

Avec spufi, faire une jointure des tables  SYSIBM.SYSTABLES   et SYSIBM.SYSCOLUMNS
pour afficher le nom des colonnes de vos tables, et d'autres informations que vous juger intéressantes.
Attention, le nom de la table est différent dans les deux tables, 
ainsi que le nom du créateur.

3) Faire un select sur les tables 
SYSIBM.SYSINDEXES  et SYSIBM.SYSPLANDEP 
pour retrouver les objets DB2 quevous avez créés
